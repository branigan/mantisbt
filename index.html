Bienvenido.

kineticalmantis es un fork del proyecto mantisbt (Mantis Bug Tracker), con el objetivo de mejorar el aspecto visual y la funcionalidad del mismo.

Dentro de las tareas principales del proyecto se encuentra:

  • Eliminar la maquetación realizada con tablas.
  • Añadir el framework de diseñado desarrollado por Twitter: BootStrap.
  • Mejorar la usabilidad
  • Migrar a HTML5
  • Incorporar un framework de javascript actual como es Jquery
  • Refactorizar ciertas partes del código fuente.
